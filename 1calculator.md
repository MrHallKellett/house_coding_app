# Problem 1 - Henry’s Mini Calculator (*)

![Henry's mini calculator](static/problems/1calculator.jpg)



Henry Kellett’s calculator broke while he was out at sea. He needs help making a new one!

## Input Format

The first line of input will be the mode of the calculator: “m” for multiply or “a” for add. The next two lines of code will contain two numbers.

## Output Format

Print the result of applying the operation with the first number and the second.

### Example 1

**Input:**

    m
    4
    8

**Output:**

    32

**Explanation:**

The first line represents multiplication, therefore the input represents the problem 4 × 8, thus the output is 32.

### Example 2

**Input:**

    a
    3
    4

**Output:**

    7

**Note:** When taking input, do not print any message. For example, `input()` or `input(““)` would be fine, but `input(“Number: “)` or `input(“Something“)` might fail the tests.
