Welcome to Kellett's First Inter-House Coding Tournament!

This is a fast-paced competition where participants go head-to-head to solve programming puzzles.

They will be helping Henry Kellett overcome the challenges he faces as he sails the high seas.


How it Works

  1. The Challenge: In each match, two participants (or teams) are given the same secret programming problem. These are logic puzzles that require writing a small computer program to solve.

  2. The Race: When the match starts, a timer begins. Participants race to write a program that correctly solves the puzzle.

  3. The Test: To win, their program must pass a series of automated tests. This means the program must produce the exact correct output for a given set of inputs, just like in the problem description's examples. The first person to submit a correct solution wins the match and advances to the next round!

Good luck to all participants!
