# Problem 5 - Henry’s Fish Farm (***)

<img src="/static/problems/3fish_farm.png" width="300"/>

Henry Kellett keeps a fish farm. He keeps a daily record of the population of his fish. On the 1st day, he had 2 fish. On the 2nd day, the two fish had a baby fish. There were now 3 fish.

Then, the fish’s population started to grow in a strange pattern. On the 3rd day, there were 5 fish. On the 4th day, there were 8 fish.

In fact, the fishes’ population continued to grow in this strange pattern:

2 3 5 8 13 21 34 ...

You are given a day. Predict and print the number of fish on that day.

## Input Format

A single integer, the required day.

## Output Format

A single integer, the population of fish on the required day.

## Example 1

### Input:

```
1
```

### Output:

```
2
```

## Example 2

### Input:

```
8
```

### Output:

```
55
```

Note: When taking input, do not print any message. For example, `input()` or `input(““)` would be fine, but `input(“Number: “)` or `input(“Something“)` might fail the tests.
