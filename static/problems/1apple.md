# Problem 2 - Henry’s Apples (*)

<img src="/static/problems/1apples.jpeg" width="300"/>

Henry Kellett has some boxes. Each box can store some number of apples. Output the total number of apples across all the boxes.

## Input Format

You are first given the number of boxes, and each following line will have the number of apples in each box.

## Output Format

A single number, the total number of apples in all the boxes.

## Example 1

### Input:

    3  
    4  
    8  
    10 

### Output:

    22

### Explanation:

The first line of the input represents that there are 3 boxes. Thus the answer is the sum of the next 3 lines (4 + 8 + 10), which equals 22. Thus the answer is 22.

## Example 2

### Input:

    2
    3
    10

### Output:

    13
