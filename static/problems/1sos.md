# Problem 20 - SOS (*)

<img src="/static/problems/1sos.jpg" width="300"/>

Henry Kellett is in a storm, he has some information about the wave in front of him. He consults the admiralty and tells them to calculate whether he can pass through the wave or not. The admiralty decides by calculating the length of the hypotenuse of the wave and see whether the ship has enough energy to pass through it.

## Input Format

Two lines of input

First: Base length

Second: Height

## Output Format

Pass if length is positive and lower than 10, No Pass otherwise.
