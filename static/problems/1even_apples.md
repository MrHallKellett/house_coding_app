# Problem 3 - Henry’s Even Apples (*)

<img src="/static/problems/1even_apples.png" width="300"/>

Henry Kellett wants to know how many of his apple boxes have an even number of apples. Output the number of apple boxes with an even number of apples.

## Input Format

You are first given the number of boxes, and each following line will have the number of apples in each box.

## Output Format

A single number, the number of boxes with an even number of apples.

## Example 1

### Input:

```
4
2
3
1
4
```

### Output:

```
2
```

### Explanation:

The first line represents the number of boxes - 4. Then the next four lines represents the number of apples in each box (2, 3, 1, 4). The first and last of these boxes have an even number of apples. Thus the output is 2.

## Example 2

### Input:

```
5
```
