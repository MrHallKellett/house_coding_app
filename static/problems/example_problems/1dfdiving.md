Example Round 1 Problem C - Depth-First Diving (*)

Captain Kellett has sent divers to explore the ocean floor in search of sunken treasure! 

Each diver has recorded the maximum depth they reached during their dive. The captain wants to know which diver went the deepest.

Given a list of diving depths (in meters), find the maximum depth reached.

Input Format

First line: An integer n (the number of divers)

Second line: n integers separated by spaces, representing the depth each diver reached (in meters)

Output Format

Print a single integer: the maximum depth reached by any diver.

Example 1

Input:

5
12 25 18 30 22

Output:

30

Explanation:

The depths are: 12, 25, 18, 30, 22 meters. The deepest dive was 30 meters.

Example 2

Input:

3
45 45 45

Output:

45

Explanation:

All three divers reached the same depth of 45 meters.

Note: When taking input, do not print any message. For example, input() or input(““) would be fine, but input(“Number: “) or input(“Something“) might fail the tests.
