Example Round 1 Problem B - Marco Polo (*)

Henry and his best friend are playing their favourite sea exploration themed game: MARCO POLO!

The rules are simple: the player that is ‘on’ shouts “MARCO?”, and the other player shouts “POLO!” until the the first player finds the second player and then shouts “IT”.

In this simulation of the game you are the second player.

Input Format

Any number of lines containing the string MARCO?

Followed by a single line containing the string IT!

Output Format

A corresponding number of lines containing the string POLO!

Example 1

Input:

MARCO?
MARCO?
IT!

Output:

POLO!
POLO!

Example 2

Input:

MARCO?
MARCO?
MARCO?
MARCO?
MARCO?
MARCO?
MARCO?
MARCO?
IT!

Output

POLO!
POLO!
POLO!
POLO!
POLO!
POLO!
POLO!
POLO!

Note: When taking input, do not print any message. For example, input() or input(““) would be fine, but input(“Number: “) or input(“Something“) might fail the tests.
