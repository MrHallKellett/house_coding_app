Example Round 1 Problem A - Ship Measuring (*) 

Henry Kellett is comparing the names of ships in his naval fleet.

Each time he writes two ship names, he wants to know which name is longer. If both names are the same length, he declares them a TIE.

For each pair of ship names:

Print the longer name.

If both names have the same number of characters, print TIE.

Input Format

The first line contains the name of the first ship

The second line contains the name of the second ship

Output Format

The name of the ship with the longer name or TIE if both ship names are equal.

Example 1

Input:

VICTORY
ENDEAVOUR

Output:

ENDEAVOUR

Example 2

Input:

ALPHA
BRAVO

Output:

TIE

Note: When taking input, do not print any message. For example, input() or input(““) would be fine, but input(“Number: “) or input(“Something“) might fail the tests.
