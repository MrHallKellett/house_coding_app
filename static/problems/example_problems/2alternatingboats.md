Example Round 2 Problem A - Alternating Boats (**)

Henry Kellett has 2 types of boats, “A” and “B”. An arrangement of boats looks like a combined version of these two characters. For example, “ABBABAA“ could be an arrangement.

He wants to know if his arrangement of boats is in alternating order. For example:

ABABAB and BABABA are alternating, but

ABBABA is not alternating.

You are given the existing boat layout. If they are arranged in order, print “Y”. Otherwise, print “N”.

Input Format

The first line contains the current layout of the ships.

Output Format

A single character, “Y” or “N” depending on if the boat layout is alternating.

Example 1

Input:

A

Output:

Y

Example 2

Input:

ABA

Output:

Y

Example 3

Input:

ABABAAABBB

Output:

N

Note: When taking input, do not print any message. For example, input() or input(““) would be fine, but input(“Number: “) or input(“Something“) might fail the tests.
