# Problem 12 (*) - Triangular Design

<img src="/static/problems/1triangular_design.png" width="300"/>

Henry Kellett really likes shapes. He especially loves triangles, but he forgot how to draw a right-angled triangle!

Help him print a right triangle made of “@“s, which is the same number of characters tall and THE SAME number of characters wide.

## Input Format

A single integer n - the width and height of the shape.

## Output Format

The required shape.

## Example 1

### Input:

```
6
```

### Output:

```
@
@@
@@@
@@@@
@@@@@
@@@@@@
```

## Example 2

### Input:

```
3
```

### Output:

```
@
@@
@@@
```

Note: When taking input, do not print any message. For example, `input()` or `input(““)` would be fine, but `input(“Number: “)` or `input(“Something“)` might fail the tests.
